#Right Amount

[![Built with Grunt](https://cdn.gruntjs.com/builtwith.png)](http://gruntjs.com/)

Help to get a number with six other number.

To test:
```bash
$ npm install -g fast-http-cli
$ fast-http -o
```

To compile:
```bash
$ npm install
$ npm install -g grunt-cli
$ grunt
```

See `Gruntfile.js` for task config.
